## 仕様

### スクリプトファイル
- 文字エンコーディング
  - Shift_JIS
- 改行コード
  - CR、LF、CRLF
- 拡張子
  - 指定なし

### 文法
基本形
```
命令の名前 引数1, 引数2, ...
```
- 先頭の空白は無視される
- 引数と引数はコンマ(,)で区切る
  - コンマの周りの空白は無視される
- 半角のセミコロン(;)でコメント
- 文字列型はダブルクオーテーションで括る  
<br>
整数リテラルは頭にドル記号($)を付けることで16進数で指定できるようになります。  
文字列リテラルのエスケープシーケンスは以下のものが利用できます。  
  
- \\n (LF)
- \\r (CR)
- \\&quot; (ダブルクオーテーション)
- \\\\ (バックスラッシュ、円記号)

### 変数
命令の引数に未初期化の変数を指定すると自動的に整数型として初期化されます。  
<br>
型
- Integer 整数(符号付き32bit)
- Float 実数(32bit精度6桁)
- String 文字列
- Handle ハンドル(32bit整数)
- Variable 変数の参照

### ラベル
アスタリスク(*)+ラベル名でラベルを置くことができます。  
ラベル名が重複した場合は最初に見つかったラベルが優先されます。
